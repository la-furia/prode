# Prode Club Libertador

Aplicación web para el pronóstico de partidos del Club Libertador (Mundial 2026).

## Características

- **Posiciones**: Tabla de clasificación con los puntos de cada jugador
- **Calendario**: Fechas y horarios de todos los partidos
- **Selecciones**: Estadísticas y pronósticos por país

## Tecnologías

- React + TypeScript
- Vite
- Tailwind CSS
- React Router (HashRouter)

## Desarrollo

```bash
npm install
npm run dev
```

## Build para producción

```bash
npm run build
```

## Despliegue

El sitio se despliega automáticamente a GitHub Pages mediante GitHub Actions.
